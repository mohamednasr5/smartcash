ربط الهاتف — اتصال حقيقي

ربط الهاتف فعلياً بدون Node.js

اتصال مباشر عبر WebUSB (كابل) أو WebSocket (نفس الواي فاي) — يعمل على GitHub Pages بالكامل.

USB — كابل مباشر
WiFi — نفس الشبكة

إعداد USB ADB (مرة واحدة فقط)

1

الإعدادات ← حول الهاتف ← اضغط رقم الإصدار 7 مرات لتفعيل خيارات المطور

2

الإعدادات ← خيارات المطور ← فعّل تصحيح أخطاء USB — هذا يسمح للمتصفح بالتواصل مع الجهاز

3

وصّل كابل USB وانتظر رسالة السماح بتصحيح الأخطاء على الهاتف — اضغط موافق وضع علامة "ثق دائماً"

4

اضغط فحص الأجهزة أعلاه — المتصفح Chrome/Edge مطلوب (يدعم WebUSB). سيطلب منك اختيار الجهاز من قائمة.

كيف يعمل WiFi Bridge؟

1

على الكمبيوتر نفسه — نزّل Python 3 (مجاني) ثم نزّل سكريبت الجسر الخفيف: cashty-bridge.py من زر "تنزيل Bridge" أدناه

2

شغّل: python cashty-bridge.py — يفتح بورت 7070 فقط (لا Node لا npm)

3

تأكد أن الهاتف والكمبيوتر على نفس الواي فاي وعليه تطبيق ADB Wireless أو وصّل كابل مع ADB WiFi مفعّل

4
